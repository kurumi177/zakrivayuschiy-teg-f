# Закрывающий тег

Интерактивный сайт-дневник для сохранения воспоминаний о пути в веб-разработке.

## Особенности

- 8 тематических карточек с этапами обучения
- Анимированные лайки с тремя состояниями (hover, active, liked)
- Уникальные CSS-фильтры для каждого изображения
- Интерактивные кнопки с hover-эффектами
- Адаптивный дизайн

## Технологии

- HTML5, CSS3 (переменные, анимации, фильтры)
- SVG-графика с CSS-анимациями
- JavaScript для интерактивности
- CSS Grid/Flexbox для вёрстки

## Структура

- Карточки воспоминаний с возможностью лайка
- Модальное окно сохранения
- Полностью адаптивный интерфейс
- Анимации на чистом CSS
